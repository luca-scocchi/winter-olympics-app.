# Winter Olympics Expert Web App

A **frontend-only web app** that acts as an expert on the **Winter Olympic Games**.  
Users can ask questions and receive detailed, engaging, and accurate answers about:

- History of the Winter Olympics  
- Statistics and medal counts  
- Records and top performances  
- Strategies, curiosities, and fun facts  

The app provides **narrative-style responses**, accessible to beginners and enthusiasts alike, citing official sources whenever possible.

## Features

- Query-based interaction with the AI expert  
- Narrative and engaging responses  
- Data-backed insights and historical context  
- Fun facts and strategic insights included  

## Tech Stack

- **HTML/CSS/JS** (frontend only)  
- AI integration via **Hugging Face Inference API** or **OpenAI Playground free tier**  
- Hosted on **GitHub Pages**  

## How to Use

1. Open the live app on [GitHub Pages](https://USERNAME.github.io/winter-olympics-app/)  
2. Type your question in the input box  
3. Press submit to receive a detailed answer from the Winter Olympics expert
