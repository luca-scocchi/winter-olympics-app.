# Usage Guide

## How to Use the App

1. Open `index.html` in your browser or access the app on GitHub Pages  
2. Type your question about Winter Olympics in the input box  
3. Click the **Submit** button  
4. The AI expert responds with:
   - Historical context  
   - Statistics and records  
   - Curiosities and fun facts  
   - Narrative storytelling  

### Example Questions

- "Who has the most medals in Winter Olympic history?"  
- "What are the strategies for winning in biathlon?"  
- "Tell me a fun fact about ski jumping."

## Hosting on GitHub Pages

1. Push the project to a GitHub repository  
2. Go to **Settings → Pages** → select branch `main` and folder `/root`  
3. Access the live app via: `https://USERNAME.github.io/REPO_NAME/`

## Updating the App

- Modify HTML/CSS/JS locally  
- Upload the updated files to the repository  
- GitHub Pages updates automatically
