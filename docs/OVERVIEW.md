# Winter Olympics Expert Web App - Overview

This web app is designed to provide detailed and engaging answers about the **Winter Olympic Games**.  
It uses a **role-prompting AI** to act as the ultimate Winter Olympics expert.

### Purpose

- Educate users about Winter Olympics history, statistics, and records  
- Provide analysis, strategies, and fun facts about events and athletes  
- Make learning accessible and engaging for both beginners and enthusiasts

### Target Audience

- Sports enthusiasts and fans  
- Students or researchers looking for accurate information  
- Beginners curious about Winter Olympics history or trivia

### Key Features

- Query-based interaction with an AI expert  
- Narrative, storytelling-style answers  
- Official sources cited whenever possible  
- Fun facts and strategic insights included
